## v2 changes

`SacredTrees.sol` is no longer part of this project. It migrated to [sacred-trees](https://github.com/sacredcash/sacred-trees)

## Dependencies

1. node 12
2. yarn
3. zkutil (`brew install rust && cargo install zkutil`)

## Start

```bash
$ yarn
$ cp .env.example .env
$ yarn circuit
$ yarn test
```
